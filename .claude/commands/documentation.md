# /documentation

## Objetivo

Atualizar documentação do projeto.

## Entrada

- Área a documentar (ex: "novo módulo financeiro", "mudança na API")
- Ou arquivo específico a atualizar

## Processo

1. Identificar documentação afetada
2. Atualizar arquivos relevantes:
   - `docs/` — documentação externa
   - `.claude/memory/` — memórias do Claude Code
   - `CLAUDE.md` — perfil e comandos
3. Verificar consistência entre documentos

## Saída Esperada

- Documentação atualizada e consistente
- Novas informações adicionadas aos arquivos existentes

## Regras

- Não duplicar informação (um fato = um lugar)
- Manter consistência: se mudar código, atualizar docs
- Links relativos funcionais
- Data da última atualização no topo
